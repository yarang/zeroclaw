## 회의 안건 및 논의 내용

### 1. 프로젝트 개요

ZeroClaw은 Rust 기반의 자율 에이전트 런타임으로, 다음과 같은 핵심 목표를 가짐:

- **고성능**: 효율적인 리소스 활용
- **고효율**: 최소한의 오버헤드로 최대 기능 제공
- **고안정성**: 안정적인 장기 실행
- **고확장성**: Trait + Factory 패턴 기반 모듈화
- **고지속가능성**: 유지보수 용이한 아키텍처
- **고보안**: Security-first 설계

### 2. 아키텍처 핵심 확장 포인트

| 모듈 | Trait | 역할 |
|------|-------|------|
| `src/providers/` | `Provider` | 모델 제공자 (OpenAI, Anthropic 등) |
| `src/channels/` | `Channel` | 통신 채널 (Telegram, Discord, Slack 등) |
| `src/tools/` | `Tool` | 도구 실행 (Shell, File, Memory, Browser) |
| `src/memory/` | `Memory` | 메모리 백엔드 (Markdown, SQLite, Vector) |
| `src/observability/` | `Observer` | 관찰가능성/모니터링 |
| `src/runtime/` | `RuntimeAdapter` | 런타임 어댑터 |
| `src/peripherals/` | `Peripheral` | 하드웨어 주변기기 (STM32, RPi GPIO) |
